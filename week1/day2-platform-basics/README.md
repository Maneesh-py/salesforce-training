# Day 2
# What is Salesforce Platform?
Salesforce Platform is a cloud-based platform that helps businesses build applications, manage customer data, automate work, and customize business processes. It provides both no-code and coding tools, making app development faster and easier.
## App
An App in Salesforce is a collection of tools, tabs, objects, and features designed for a specific business process.
## Object
An Object is like a table in a database that stores information.
## Tab
A Tab helps users access objects, records, and features easily from the navigation bar.
# Difference Between Configuration and Coding
Configuration means customizing Salesforce using clicks, drag-and-drop tools, and settings without writing code. Coding means using programming languages like Apex or Lightning Web Components to build advanced functionality and custom logic.
Configuration is easier and faster for basic tasks, while coding is used for complex business requirements.

Task 3: Thinking Task

When do we use Configuration (No Code)?

Configuration is used when we can customize Salesforce using clicks and settings without writing code.

Example:
Creating custom fields, Building flows and page layouts

When do we use Coding (Apex)?

Coding is used when the requirement is complex and cannot be handled using simple configuration tools.

Example:
Sending custom automated logic, Integrating Salesforce with external applications
